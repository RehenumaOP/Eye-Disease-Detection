👁️ Eye Disease Detection Using Deep Learning
📌 Project Overview

This project focuses on detecting and classifying eye diseases using Machine Learning and Deep Learning techniques.
The system analyzes eye images and predicts diseases using advanced neural network architectures such as CNN, VGG16, and MobileNetV2.

The project demonstrates practical applications of Artificial Intelligence in the healthcare sector.

🎯 Objectives
Detect eye diseases from medical images
Compare Machine Learning and Deep Learning models
Improve classification accuracy using transfer learning
Explore AI applications in healthcare
